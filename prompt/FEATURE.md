ทำเว็บเอาไว้จองคิวสำหรับร้านเกม Racing Simulation

เน้นใข้งานง่าย ไม่ต้องมีฟีเจอร์เยอะ

โดยเอาพื้นฐานก็พอ

ให้ใช้ supabase สำหรับ database เก็บแค่ customers และ queues ก็พอ เน้น simple 

1. ลูกค้า group router /customer
- ให้สามารถ เลือก เครื่องที่จะจองคิวได้
- ดูสถานะของคิว

2. admin group router /backend
- ดูสถานะของคิว
- จัดการคิว
- และอื่นๆที่จัดการกับ db customers และ queues


ห้ามทำฟีเจอร์อื่่นอกเหนือจากนี้ เพราะต้องการทำโปรเจคให้เร็วที่สุด

ก่อนจะสร้าง migrations ให้ศึกษา migrations ที่มีอยู่ก่อน
